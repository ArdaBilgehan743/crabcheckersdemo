# Blender Sources

Place professional `.blend` source files here.

The current generated Blender sources are written to `assets/blender/source/` by:

```bash
npm run assets:blender
```

Expected first production sources:

- `characters/patient-bear.blend`
- `characters/the-hat.blend`
- `characters/lantern-keeper.blend`
- `tokens/crab-tokens.blend`
- `props/table-room.blend`
- `props/board-6x6.blend`

Export final runtime `.glb` files to `public/models/`.
